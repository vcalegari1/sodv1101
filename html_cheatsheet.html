HTML Cheat Sheet

    • <!DOCTYPE html>: Declares the document type and version of HTML.
    • <html>: The root element of an HTML document.
    • <head>: Contains meta-information about the HTML document, such as character set and viewport settings.
    • <meta>: Provides metadata about the HTML document, like character set and viewport.
    • <title>: Defines the title of the HTML document, displayed in the browser tab.
    • <link>: Links an external stylesheet to the HTML document.
    • <body>: Contains the content of the HTML document.
    • <header>: Represents a group of introductory or navigational aids.
    • <h1>: Defines a top-level heading.
    • <nav>: Represents a navigation section.
    • <ul>: Defines an unordered (bulleted) list.
    • <li>: Defines a list item.
    • <div>: Defines a division or a section in an HTML document. Often used for grouping and applying styles.
    • <p>: Defines a paragraph.
    • <img>: Embeds an image in the document.
    • <span>: Defines an inline section in an HTML document. Similar to <div>.
    • <br>: Inserts a line break.
    • <strong> or <b>: Represents strong importance, typically displayed as bold.
    • <em> or <i>: Represents emphasized text, typically displayed as italic.
    • <footer>: Represents a footer section.
    • <script>: Embeds or references an external JavaScript file.
    • <a>: Defines a hyperlink.
